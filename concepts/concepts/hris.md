# HRIS

Hudson Recursive Interaction System (HRIS) is a framework for studying long horizon interaction between humans and large language models.

Rather than treating prompts as isolated events, HRIS studies how repeated interaction patterns shape inference trajectories over time.

The framework focuses on:
- reconstructive continuity,
- semantic basin dynamics,
- interaction shaped reasoning,
- and trajectory stabilization effects.

HRIS treats interaction itself as a primary unit of analysis rather than isolated prompts or static outputs.
